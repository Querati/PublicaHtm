As estruturas de habilidades fornecem uma visão dos níveis de competência necessários para funções específicas. Else definem:<br/><br/><ul>
<li>As funções dentro de uma área de trabalho</li><li>As habilidades exigidas para cada função</li><li>A profundidade de conhecimento necessária para desempenhar a função com sucesso</li></ul>
<br/>Else são relativamente comuns para definir as habilidades necessárias para uma atribuição de consultoria e/ou gerenciamento de projetos, para entregar um projeto ou pacote de trabalho específico. Else também são amplamente utilizados por agências de recrutamento e pesquisa para combinar candidatos e funções.<br/><br/>Seu valor deriva de sua capacidade de fornecer um meio de identificar rapidamente correspondências e lacunas de habilidades. Aplicados com sucesso, else podem garantir que os candidatos estejam aptos para os cargos que lhes forem atribuídos.<br/><br/>Seu valor no contexto da Arquitetura Corporativa surge da imaturidade da disciplina de Arquitetura Corporativa e dos problemas que surgem a partir disso.<br/>
